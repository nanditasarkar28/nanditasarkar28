## Featured Projects

### 💼 Jobster — Full-Stack Job Tracking Platform
> React · Node.js · Express · MongoDB · JWT · REST APIs

A full-stack job application tracker with a separate Frontend and Backend. Users can add, update, and manage job applications in one place. Built with a React frontend and a Node/Express backend with MongoDB as the database, secured with JWT auth.

Clean architecture — the kind where the frontend and backend don't need to know much about each other to work together.

[![View Repo](https://img.shields.io/badge/View_Repo-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/nanditasarkar28/Jobster)

### ✅ TaskManagerAPI — REST API for Task Management
> Node.js · Express.js · MongoDB · REST APIs · Vercel

A production-deployed REST API for managing tasks — built with a clean layered architecture (controllers, routes, middleware, models). Handles custom error responses gracefully and is live on Vercel.

The project structure here is what I'd call "real backend layout" — not just an index.js with everything dumped in.

### 🔐 JWT — Authentication from Scratch
> Node.js · Express.js · JWT · MongoDB · Middleware

A dedicated implementation of JWT-based authentication — register, login, protected routes, token verification middleware. Built this separately to understand auth deeply before relying on libraries to do it silently.

If you want to understand how authentication actually works end-to-end, building it once from scratch is the best way.
